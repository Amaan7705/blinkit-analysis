# 🛒 Blinkit Sales Data Analysis

## 📌 Overview
This project analyzes Blinkit grocery sales data using Python to extract meaningful business insights.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Key Analysis
- Total Sales & Average Sales
- Number of Items Sold
- Average Rating
- Sales by Item Type
- Sales by Outlet Size & Location

## 💡 Insights
- Tier 3 outlets have highest sales
- Regular items contribute more revenue
- Fruits and Snacks categories dominate

## ▶️ How to Run
1. Open Jupyter Notebook
2. Run all cells
